** TextUtils
TextUtils is a React-based web application that helps users perform various text operations quickly and efficiently.

**Features
Convert text to UPPERCASE
 Convert text to lowercase
 Remove extra spaces
 Copy text to clipboard
  Count words and characters
 Estimate reading time
 Live text preview

** Technologies Used
 React.js
 JavaScript
 HTML
 CSS
 Bootstrap

** How to Run
1. Clone the repository
2. Install dependencies
